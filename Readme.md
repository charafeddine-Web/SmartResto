# SmartResto
Le projet consiste à développer une application qui utilise Angular et NgRx pour résoudre certains des problèmes récurrents rencontrés dans la gestion quotidienne d'un restaurant. Une vidéo d'introduction mettra en évidence 10 problèmes clés liés à la gestion des restaurants, qui nuisent à l'expérience client et à l'efficacité opérationnelle.
